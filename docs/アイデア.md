# V-chat

ビデオ通話を行うwebアプリケーション

通常のビデオ通話と異なる点は画面に映る姿が人ではなく3Dキャラモデルということであり、ユーザーは顔を隠しながらノンバーバルコミュニケーションを行うことができる

人との繋がりが少なくなってきている現代人が抱えている心の閉鎖感を打破し、カジュアルな人との交流を推進する。

顔出しが怖い人にとって気軽に人と繋がれる機会を増やしたいという目的がある

## 機能想定

- ユーザー認証
  - email
  - Github
  - Googleなど
- ユーザー情報登録
  - ユーザーID
  - ユーザー名
  - 性別
  - V体のID？
  - created_at
- マッチング機能(1対１ ランダムorカスタム[番号で部屋を作る])
- ビデオ通話[相手のV体の動き、音声]
- 自分のV体をVroidHubから設定する
- ユーザー情報を修正する
- 新規ユーザー登録をする
- ログインする

## 用語

V体：ユーザーが使用する3Dキャラモデル

ランダムマッチング：ランダムなユーザーとマッチングするようにする

セレクトマッチング：ルームを作成してIDを発行、相手にIDを渡し、マッチングできるようにする

## 使用技術(想定)

### フロントエンド

- Next.js
- React Three Fiber (VRM3dアバター描写)
- MediaPipe holisic (V体の動き検知？)
- VroidSDK (V体のモデルを提供してくれる想定[.vrm])
- shadcn/ui + TailwindCSS
- LiveKitSDK (ビデオ通話)

### バックエンド

- Vercel Edge Functions (JWT 発行・プロフィール CRUD)
- Neon PostgreSQL Free (ユーザー／セッション永続化)
- Upstash Redis Free (ランダムマッチ待機キュー)
- LiveKit Cloud Build (音声 SFU／5 k min 無料)
- Cloudflare R2 (VRM ファイル保管・外向き egress 0 円)

### インフラ運用
- Vercel Hobby (フロント＋API ホスティング／帯域 100 GB まで無料)
- GitHub Actions (テスト → vercel deploy CI/CD)
- 無料枠 SaaS 組み合わせ (月額ほぼ $0 で PoC 運用)
- スケールアップ手順 (需要に応じて LiveKit Ship, Neon Launch へ昇格)
- 監視：Vercel Insights と LiveKit Dashboard（基本無料）

## 将来的な展望・発展

- その他オンライン会議アプリの拡張機能としてV体で参加できるようにする
- ビデオ回数に対してマネタイズ
