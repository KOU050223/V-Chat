# ゲームルール

### 準備フェーズ
- 各プレイヤーは自動で以下が割り当てられる
- アバター（Vroid）
- 役職（人狼・村人・占い師など）
- キャラ設定（例:「高貴なお嬢様」「気弱な少年」「狂気の科学者」）

### ゲーム進行
Bot（GM代行)がフェーズを管理する
1. 導入（自己紹介）

割り当てられた設定になりきって自己紹介

2. 議論フェーズ

アバターで演じつつ人狼を探す

3. 投票フェーズ

各自が投票、最多票のプレイヤーが退場　自動ミュート

4. 夜フェーズ

人狼が襲撃先を決定(人狼専用のテキスト機能で会話可能)／占い師が占う　自動ミュート　

5. 昼フェーズ

生存者で再び議論

###  脱落後の処理

- 脱落者は観戦モード（聞き専）になる
- もしくはテキストチャットで雑談（本編に干渉できない）

### 勝敗条件
- 村人陣営: 人狼を全滅させる
- 人狼陣営: 村人の数を人狼以下にする

### 勝利条件１
- ゲームの勝敗と物語の勝敗の両方達成で完全勝利条件
- ゲームの勝敗：村人陣営 or 人狼陣営が勝つ 
- 物語の勝敗：ゲームの勝敗後、参加者の人物設定が公開されて、指定されたキャラ設定になりきれていたのか投票。投票率が8割で勝利
- 一番投票率が高かった人にベストアクター賞など称号がもらえる。

### 勝利条件２
- ゲームの勝敗と物語の勝敗の両方達成で完全勝利条件
- ゲームの勝敗：村人陣営 or 人狼陣営が勝つ 
- 物語の勝敗：個人が設定されたキャラ目標を達成できたかどうか  

各プレイヤーは「役職」とは別に、キャラ設定や個人目標が与えられる。  
この目標を達成すると、敗北しても「物語上の勝利」になる。

例「忠実な執事（人狼）」 → 主人（指定されたキャラ）を最後まで守れれば追加勝利。  
　「謎の旅人（村人）」 → 最後まで正体を隠し通す（＝自分の本当の設定をばらさない）と追加ポイント。
