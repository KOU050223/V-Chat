## 🚀 クイックスタート

1. **リポジトリのクローン**

   ```bash
   git clone https://github.com/KOU050223/V-Chat.git
   cd V-Chat
   ```

2. **依存関係のインストール**

   ```bash
   npm install
   ```

3. **環境設定**
   - [Firebase設定ガイド](../Firebase設定.md) に従ってFirebaseを設定
   - `.env.local` ファイルを作成し、設定値を入力

4. **開発サーバーの起動**

   ```bash
   npm run dev
   ```

## 🛠️ 技術スタック

- **フロントエンド**: Next.js 15, React 19, TypeScript
- **UI/スタイリング**: Tailwind CSS, shadcn/ui
- **認証**: Firebase Authentication
- **3Dモデル**: Three.js (予定)
- **リアルタイム通信**: WebRTC (予定)
